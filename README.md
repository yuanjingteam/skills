# Skills 集合

本仓库包含各种开发任务的有用技能集合。

## 可用技能

| Skill 名称         | 描述                                                                                                                                                                             | 使用场景                                                                                                                                           |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| [nvm-auto](nvm-auto/) | 自动化配置 Node 版本：检测项目所需版本，优先使用同大版本已安装的最高小版本，缺失则安装并生成 .nvmrc。每次执行自动检查并安装 shell hook（zsh/bash），之后 cd 进项目自动切换版本。 | 当用户提到 nvm、Node 版本、.nvmrc、进入新项目需要配置环境时，使用此技能。**直接执行 `bash nvm-auto.sh`，不要重写脚本逻辑或逐步执行命令。** |

## 安装

每个技能都可以独立安装和使用。请参考各个技能的文档了解具体的安装说明。
